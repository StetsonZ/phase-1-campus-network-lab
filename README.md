# Multi-VLAN Campus Network Lab – Phase 1

## Overview

This project documents the design, implementation, and troubleshooting of a small campus network built using Cisco routing and switching technologies.

Phase 1 focused on:

- VLAN segmentation
- Layer 2 trunking
- Layer 3 inter-VLAN routing
- DHCP services
- Router-on-a-stick style edge connectivity
- End-to-end connectivity validation
